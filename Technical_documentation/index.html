What is JavaScript?

JavaScript is a versatile, high-level programming language primarily used for web development.

It enables interactive web pages and is an essential part of web applications.

JavaScript can also be used on the server-side (e.g., with Node.js).

Originally created in 1995, JavaScript has evolved significantly over the years.

Today, it's one of the most popular programming languages in the world.

Variables

Variables are used to store data values in JavaScript.

In JavaScript, you can declare variables using let, const, or var.

The let keyword allows you to declare block-level variables.

The const keyword is used for variables that shouldn't be reassigned.

The var keyword is the older way to declare variables.

  • let: Used to declare variables that can change.
  • const: Used to declare constants that cannot be reassigned.
  • var: Older syntax, less commonly used in modern JavaScript.
let name = "Alice"; // This can be changed
const age = 30; // This cannot be changed
var city = "New York"; // Older syntax
Data Types

JavaScript has several fundamental data types.

These data types define the kind of values you can work with.

JavaScript is dynamically typed, meaning variables can hold any type.

The typeof operator can help determine a variable's type.

Understanding data types is crucial for effective programming.

  • String: Text data, enclosed in quotes.
  • Number: Numeric data (integer or floating-point).
  • Boolean: Represents true or false.
  • Object: A collection of key-value pairs.
  • Array: A list of values.
  • Null: Represents an intentional absence of any value.
  • Undefined: A variable that has been declared but not assigned a value.
let message = "Hello, world!"; // String
let score = 95; // Number
let isActive = true; // Boolean
let person = { name: "Alice", age: 30 }; // Object
let colors = ["red", "green", "blue"]; // Array
let emptyValue = null; // Null
let uninitialized; // Undefined
Operators

JavaScript operators perform operations on variables and values.

Operators are essential for performing calculations and comparisons.

JavaScript has several categories of operators.

Operator precedence determines the order operations are performed.

Understanding operators is key to writing effective JavaScript code.

  • Arithmetic Operators: +, -, *, /, %
  • Comparison Operators: ==, ===, !=, !==, <, >, <=, >=
  • Logical Operators: && (AND), || (OR), ! (NOT)
let sum = 10 + 5; // 15
let isEqual = (5 === 5); // true
let isTrue = (true && false); // false
Control Structures

Control structures manage the flow of program execution.

They allow your code to make decisions and repeat actions.

JavaScript provides several control structures.

Proper use of control structures makes code more efficient.

They are fundamental to all programming languages.

  • If Statements: To execute code conditionally
    • if (age >= 18) {
    console.log("Adult");
} else {
    console.log("Minor");
}
  • Switch Statements: To execute code based on different cases.
    • let day = 2;
switch (day) {
    case 1:
        console.log("Monday");
        break;
    case 2:
        console.log("Tuesday");
        break;
    default:
        console.log("Another day");
}
  • Loops: To repeat code.
    • // For Loop
for (let i = 0; i < 5; i++) {
    console.log(i);
}

// While Loop
let count = 0;
while (count < 5) {
    console.log(count);
    count++;
}
Functions

Functions are reusable blocks of code that perform specific tasks.

They help organize code into logical, reusable pieces.

Functions can accept parameters and return values.

JavaScript functions are first-class objects.

Modern JavaScript also supports arrow functions.

function greet(name) {
    return "Hello, " + name + "!";
}
console.log(greet("Alice")); // "Hello, Alice!"
